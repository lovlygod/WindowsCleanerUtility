# Руководство по внесению вклада

Благодарим вас за интерес к нашему проекту! Мы рады любому вкладу в развитие Windows Cleaner Utility.

## Как внести вклад

### Сообщить об ошибке

Если вы нашли ошибку, пожалуйста, создайте [issue](https://github.com/lovlygod/WindowsCleanerUtility/issues/new?template=bug_report.md) с подробным описанием проблемы.

### Предложить новую функцию

Если у вас есть идея для новой функции, создайте [issue](https://github.com/lovlygod/WindowsCleanerUtility/issues/new?template=feature_request.md) с описанием.

### Внести изменения в код

1. Форкните репозиторий
2. Создайте новую ветку для ваших изменений (`git checkout -b feature/ВашаФункция`)
3. Внесите изменения
4. Убедитесь, что все тесты проходят
5. Зафиксируйте изменения (`git commit -m 'Добавить новую функцию'`)
6. Отправьте изменения в ваш форк (`git push origin feature/ВашаФункция`)
7. Создайте Pull Request

## Стиль кодирования

- Следуйте стандартам кодирования C#
- Используйте PascalCase для имен классов и методов
- Используйте camelCase для переменных
- Пишите понятные имена переменных и методов
- Добавляйте комментарии к сложным участкам кода

## Тестирование

- Пишите unit-тесты для новых функций
- Убедитесь, что все существующие тесты проходят
- Проверяйте работу вручную перед отправкой изменений

## Структура проекта

- `Services/` - сервисы для различных видов очистки
- `Properties/` - свойства проекта
- `Documentation/` - документация
- `image/` - изображения
- `logs/` - логи приложения
